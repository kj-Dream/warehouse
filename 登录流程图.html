🔐 登录模块完整流程图

阅读说明： 本图按时间顺序从上到下。每列是一个不同的层级，箭头指向表示"调用了什么"。 你输入的 admin/123456 从浏览器出发，经过 4 个层级，最终拿到一个 JSON 响应。
🌐 浏览器（前端）
Login.vue
用户输入 admin / 123456
点击"登录"按钮
⬇
axios 发送请求
POST /api/auth/login
{"username":"admin", "password":"123456", "captcha":8}
⬇
📍 请求到达后端（vite代理转发到8080）
⬇
🎯 Controller 层
LoginController.java
@PostMapping("/login")
⬇
Step 1：参数校验
username 和 password 不能为空
⬇
Step 2：验证码校验
从 Session 取出正确答案比对
session.getAttribute("captchaAnswer")
⬇
Step 3：调 Service
loginService.login(request, session)
➡ 调用
⚙️ Service 层
LoginServiceImpl.java
接收 Controller 传过来的
request 和 session
⬇
Step 1：查用户
sysUserMapper.findByUsername("admin")
➡ 调 Mapper
Step 2：Mapper 去 XML 找 SQL
namespace + id 匹配
SysUserMapper.java → SysUserMapper.xml
➡ 执行 SQL
🗄️ Mapper + 数据库
SQL 执行
SELECT * FROM sys_user
WHERE username = 'admin'
⬇
数据库返回结果
SysUser 对象：
{id:1, username:"admin",
password:"$2a$10$...", status:1, roleId:1}
⬆ 返回
⬆ 返回流程（从下往上）： 数据库返回 SysUser → Mapper 封装好 → Service 层接着处理剩下的业务逻辑
⚙️ Service 继续
Step 3：比对密码
PasswordUtil.matches("123456", "$2a$10$...")
结果是 true ✅
⬇
Step 4：检查账号状态
user.getStatus() == 1 → 已启用 ✅
⬇
Step 5：查角色
sysRoleMapper.findById(1)
→ {roleName:"管理员", roleKey:"admin"}
⬇
Step 6：封装返回
把 SysUser 的字段复制到 LoginResponse
注意：password 不复制！
⬇
Step 7：存入 Session
session.setAttribute("user", response)
⬆ 返回 Controller
🎯 Controller 返回
收到 Service 的 LoginResponse
return Result.success("登录成功", response)
⬆ 返回浏览器
🌐 前端收到响应
axios 收到 JSON
code=200 → 登录成功
用户信息存入 localStorage
⬇
跳转到首页仪表盘
router.push('/dashboard')
📦 最终返回给前端的数据
{
"code": 200, // 200=成功，500=失败
"msg": "登录成功", // 提示信息
"data": {
"id": 1,
"username": "admin",
"realName": "系统管理员",
"role": { "roleName": "管理员" } // 角色信息（决定侧边栏菜单）
},
// ⚠️ 没有 password 字段！
}

📌 关键代码文件：

Controller 层 → warehouse/src/main/java/.../controller/LoginController.java 第84行
Service 层 → warehouse/src/main/java/.../service/impl/LoginServiceImpl.java 第64行
Mapper 接口 → warehouse/src/main/java/.../mapper/SysUserMapper.java 第41行
SQL 语句 → warehouse/src/main/resources/dao/SysUserMapper.xml 第70行
DTO → LoginRequest.java / LoginResponse.java / Result.java
前端页面 → warehouse-web/src/views/Login.vue
